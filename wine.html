Installation Instructions for Wine 4.0

There comes a point in time where you may need to install Wine in order to run Windows EXE Executable Files.

Here are the instructions to install it:

  • You will need to enable 32-Bit architecture

  • Run sudo dpkg --add-architecture i386

  • wget -qO - https://dl.winehq.org/wine-builds/winehq.key | sudo apt-key add -

  • For Ubuntu 19.04 run sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ disco main'

  • For Ubuntu 18.04 run sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ bionic main'

  • For Ubuntu 16.04 run sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ xenial main'

  • sudo apt-get update

  • sudo apt-get install --install-recommends winehq-stable

  • sudo apt-get install aptitude

  • sudo aptitude install winehq-stable

  • wine --version

  • Run it by going into the folder the .exe is in and use wine filename.exe
